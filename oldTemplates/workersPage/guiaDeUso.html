Guía de uso del gestor de herramientas

El gestor de herramientas (se encuentra en "Herramientas") permite asegurar la trazabilidad y disponibilidad de las diferentes máquinas y herramientas del taller.

Conceptos:
Máquina: Las máquinas hacen uso de las diferentes herramientas para la producción de piezas.
Herramienta: Las piezas usadas por lasm máquinas. Tienen distintos atributos (como diámetro de corte, pedido a proveedor...).
Arquetipo: Un arquetipo sirve para generar popular rápidamente los datos de las herramientas. (Más información más adelante)
El proceso de gestión:
Aquí se demostrará un ejemplo del proceso de gestión de herramientas. Lea atentamente ya que contiene conceptos muy importantes.
1)Crear pedido a proveedor
Desde el administrador (/admin) cree un proveedor. Entonces cree un pedido a proveedor (creamos el proveedor primero para así asignarle el pedido).
2)Cree una máquina
Desde el administrador (/admin) cree una máquina.
3)Cree un arquetipo
Desde /herramientas/arquetipo cree un arquetipo. Asigne los datos necesarios (para cada máquina en la base de datos aparecerá un S y F).
4)Cree una máquina
Desde /herramientas cree una herramienta. Puede elegir el arquetipo anteriormente creado, vea cómo los datos son populados con la información del arquetipo, recuerde de asignar una fecha de entrada, pedido a proveedor (como el que creamos anteriormente) y la máquina a la que estará destinada la herramienta. En el campo "extras" puede añadir información en el formato json.
Cosas a recordar:
  • Al crear una máquina, los arquetipos anteriores no tienen asignado el S y F a la máquina.
